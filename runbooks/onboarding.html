Onboarding Runbook

A practical checklist you can follow immediately.

Checklist

  • Define one agent job with a measurable outcome.
  • Limit tools to only what’s required (least privilege).
  • Create a clear input template and output format contract.
  • Run 10 test cases and document failures.
  • Add logging + retry/timeout policy, then iterate weekly.
