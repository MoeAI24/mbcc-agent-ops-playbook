Onboarding Runbook

A practical checklist you can follow immediately.

Checklist

  • Define one agent job with a measurable outcome.
  • Limit tools to only what’s required (least privilege).
  • Create a clear input template + output contract.
  • Run 10–20 test cases and document failures.
  • Add logs + retry/timeout policy, then iterate weekly.
